# Strategy Canvas — Logistics & Transport

## Blue Ocean ERRC Grid

| Eliminate | Reduce |
|-----------|--------|
| Manual paper-based tracking (all digital) | Delivery exceptions (from "lost in system" to real-time tracked) |
| Hidden delivery charges (dimensional weight surprises) | COD settlement time (from 7–14 days to 72 hours) |
| "Driver called customer" — failed delivery tagged as "attempted" | Parcel damage rate (from 3–5% to <1%) |
| Terminal handling multiple times (direct routing) | Empty return trips (match return loads) |

| Raise | Create |
|-------|--------|
| Tracking transparency (real-time, WhatsApp update) | API-first logistics for e-commerce (plug-and-play) |
| Delivery speed (same-day Dhaka standard) | Subscription delivery plan for merchants |
| Driver professionalism (uniform, training, rating) | Live freight marketplace (Uber-for-trucks platform) |
| COD reliability (72-hour settlement) | Return/reverse logistics as a service |

# Regulatory & Compliance — BIM Global Outreach (iNCOMPLEX)

## BIM Mandates by Target Market

| Country | BIM Mandate Status | Key Requirement |
|---------|--------------------|-----------------|
| United Kingdom | BIM Level 2 mandated on all government-funded projects (2016) | PAS 1192 → ISO 19650 transition |
| Singapore | BIM mandated on all new building projects >20,000 sqm (2015) | BCA BIM Guide, CORENET |
| USA (GSA) | BIM required on all GSA-funded projects | GSA BIM Guide, NBIMS-US |
| UAE (Dubai) | BIM mandated on all buildings >40 floors (2014), expanded to all >20 floors (2024) | DM BIM Guide, ISO 19650 |
| Saudi Arabia | BIM increasingly required for Vision 2030 / NEOM projects | NEOM BIM Framework |
| Australia | No federal mandate, but state-level adoption (NSW, QLD, VIC active) | ISO 19650 adoption underway |
| Canada | No federal mandate, but provincial adoption growing | ISO 19650 adoption |
| Germany | No mandate, but public projects increasingly request BIM | VDI 2552 / BIM Germany |
| Netherlands | Strong BIM adoption (not mandated) | RGD BIM Norm |
| Ireland | Growing adoption, encouraged on public works | ISO 19650-aligned |

## ISO 19650 Compliance

All BIM deliverables should be ISO 19650-aligned:
- **Part 1:** Concepts and principles
- **Part 2:** Asset delivery phase
- **Part 3:** Asset operational phase
- **Part 4:** Information exchange
- **Part 5:** Security-minded approach
